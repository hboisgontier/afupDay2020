Bonne pratique n° 66 : Utiliser la simple cote ' au lieu du guillemet "

Bonne pratique n° 68 : Libérer de la mémoire les variables qui ne sont plus nécessaires (unset())

Bonne pratique n° 69 : Ne pas appeler de fonction dans la déclaration d’une boucle de type for

Le code suivant
for($i = 0; $i<getNbElements(); ++$i) {
    …
}
doit être remplacé par
$nb = getNbElements();
for($i = 0; $i<$nb; ++$i) {
    …
}

C’est un ensemble de petites choses qui mises bout à bout améliorent l'emprunte carbone. 🌍

Les sites rapides et efficaces sont mieux référencés.
